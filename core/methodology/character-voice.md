# The Character Voice Method

> **CANONICAL METHODOLOGY** -- This file is the single source of truth for character voice design.
> A non-Kiro user can read ONLY this file and create a complete vocal identity for any character,
> from accent through instrumentation through mode selection.
> For full accent/instrumentation/mode tables, see `references/CHARACTER_VOICE_REFERENCE.md`.

---

## Core Principle

> An accent is not just how words sound -- it is a COMPLETE WORLDVIEW expressed through vowels,
> consonants, rhythm, and cultural association. When you write for an accented voice, you are
> writing for an entire musical ecosystem.

---

## What Accent Determines

1. **Vowel placement** -- which notes sustain well
2. **Consonant weight** -- which rhythms feel natural
3. **Rhythmic phrasing** -- how words fall on beats
4. **Cultural instrumentation** -- what sounds "belong"
5. **Genre expectations** -- what the audience expects
6. **Lyric vocabulary** -- dialect words, colloquialisms
7. **Emotional register** -- how feelings are expressed culturally
8. **Mode/scale** -- harmonic language matching the culture

---

## The Complete 11-Step Voice Design Procedure

### Step 1 -- Identify the Speaker

Answer these questions:
```
Who: [name/identity]
What are they: [human? creature? entity? concept?]
Age/era: [young? ancient? timeless?]
Emotional state: [at the START of the song, or existential state for non-human entities]
Cultural origin: [real-world basis / fantasy world / pre-cultural / cosmic / elemental]
Vocal type: [singing / speaking / non-vocal (expressed through instrumentation/production)]
```

**If "non-vocal":** This character is expressed through sound design, instrumentation, and production rather than literal voice. Steps 3, 4, and 9 will need adaptation (see non-vocal variant below).

### Step 2 -- Choose Accent Base / Sonic Character

**For real-world characters:** Pick the regional accent that matches their identity.

**For fantasy characters:** Choose a BLEND of real-world phonetic models:
```
Primary accent model: [e.g., Scottish]
Secondary influence: [e.g., Nordic]
Tertiary texture: [e.g., Semitic consonant weight]
```

**For non-human entities:** Choose by FEELING, not geography:
```
What does this presence FEEL like? -> [e.g., stone, wind, fire, water, void, pressure]
What real-world sounds share that quality? -> [e.g., whale song, tectonic rumble, ice cracking]
```

### Step 3 -- Define Vowel Character

| Question | Answer Guide |
|---|---|
| Open or closed vowels? | Open = warm/vast, closed = tight/urgent |
| Long or short? | Long = sustained/patient, short = percussive/urgent |
| Warm or cold? | Warm = human/close, cold = distant/alien |
| Which vowels sustain best? | List 2-3: "ah", "oh", "ee", etc. |
| Which to avoid sustaining? | List 1-2 |

**Vowel placement by accent (abbreviated):**

| Accent | Best Sustained Vowels | Avoid Sustaining |
|---|---|---|
| Southern US | "ah", "ay", "oh" | Short "i" |
| Irish | "oh", "ah", "ee" -- all musical | -- |
| Scottish | "ah", "oh", rolled "r" can sustain | Short vowels |
| Nordic | Clear "a", "e", "o" | Nasal vowels |
| Elvish (fantasy) | All sustain beautifully | -- |
| Dwarvish (fantasy) | Short vowels -- do NOT sustain | Long flowing vowels |

See `references/CHARACTER_VOICE_REFERENCE.md` for the complete vowel table.

### Step 4 -- Define Consonant Character

| Question | Answer Guide |
|---|---|
| Hard or soft? | Hard = aggressive/grounded, soft = flowing/ethereal |
| Special features? | Rolled R? Glottal stops? Sibilants? |
| Consonant clusters? | Few = singable/flowing, many = rhythmic/percussive |
| Plosives present? | Yes = punchy, no = flowing |

For non-vocal entities: describe the character of attacks/transients in the representing sound.

### Step 5 -- Define Rhythmic Tendency

Where does this voice sit relative to the beat?

| Position | Feels Like | Character Types |
|---|---|---|
| Behind the beat | Lazy, exhausted, ancient, vast | Dwarves, wounded entities, Southern characters, geological presences |
| On the beat | Grounded, deliberate, inevitable | Warriors, mentors, Nordic, keepers, duty-bound |
| Ahead of the beat | Urgent, anxious, eager | Young characters, city dwellers, desperate/panicked |
| Floating/rubato | Ethereal, timeless, untethered | Elves, spirits, cosmic entities, void presences, dream-logic beings |

### Step 6 -- Fill the Character Voice Template

**For vocal characters:**
```
CHARACTER VOICE:
- Name/Identity:
- Accent base:
- Vowel character:
- Consonant character:
- Rhythmic tendency:
- Register: [bass/baritone/tenor/alto/soprano + specific range]
- Texture: [gravelly/smooth/breathy/raspy/clear/cavernous/etc.]
- Emotional subtext: [what the voice FEELS like underneath]
- Dialect words: [5-10 words to use in lyrics that reinforce the accent]
- Instruments that match:
- Production space: [where does this voice live -- cave/forest/battlefield/void?]
- Mode/scale: [what harmonic language matches this culture?]
```

**For non-vocal entities (Character Sonic Presence):**
```
CHARACTER SONIC PRESENCE:
- Name/Identity:
- Sonic character: [what it FEELS like -- replaces "accent base"]
- Vowel character: [of the sonic proxy, if applicable]
- Consonant character: [of the sonic proxy / attack transients]
- Rhythmic tendency:
- Frequency range: [replaces "register" -- e.g., sub-bass <40Hz + choir mid-range]
- Timbral texture: [expanded -- e.g., sub-bass rumble, glass resonance, reversed whale song]
- Emotional/existential subtext: [what the presence FEELS like]
- Dialect words: N/A -- entity does not use language
- Instruments that manifest it:
- Production space: [where does this entity exist -- ocean trench? cathedral? void?]
- Mode/scale:
- Exclusions: [what this entity NEVER does -- e.g., speaks, uses words, produces melody]
```

### Step 7 -- Select Matching Instrumentation

Based on the accent/culture, choose instruments that BELONG:

| Culture | Primary Instruments | Secondary |
|---|---|---|
| Southern US | Acoustic guitar, pedal steel, fiddle | Lap steel, harmonica |
| Irish/Celtic | Fiddle, tin whistle, bodhran | Uilleann pipes, harp |
| Scottish | Bagpipe, fiddle, clarsach | Smallpipes, drums |
| Nordic/Viking | Tagelharpa, war drums, hardingfele | Nyckelharpa, chant |
| Japanese | Koto, shamisen, taiko | Shakuhachi, temple bells |
| Dwarvish (fantasy) | Deep drums, anvil, war horns, bagpipes | Stone percussion, chant |
| Elvish (fantasy) | Harp, flute, strings, bells, choir | Woodland ambience |
| Industrial/Cyber | Synths, electronic percussion, sub bass | Static, machine hum |

See `references/CHARACTER_VOICE_REFERENCE.md` for the full instrumentation tables.

If the character's ecosystem does not have obvious instruments, build from the FEELING:
```
Voice feels like: [stone? wind? fire? water? machine?]
-> Instruments that evoke that: [deep drums? flutes? distortion? ambient pads?]
```

### Step 8 -- Choose Mode/Scale

The harmonic language should match the cultural DNA:

| Culture | Natural Modes | Feel |
|---|---|---|
| Irish/Celtic | Dorian, Mixolydian, Aeolian | Modal, folk, open fifths |
| Scottish | Pentatonic minor, Aeolian | Bagpipe drones + pentatonic |
| Nordic | Aeolian, Phrygian, Locrian | Dark, modal, ancient |
| Southern US | Mixolydian, major pentatonic | Bluesy, warm |
| Middle Eastern | Phrygian dominant, maqam | Exotic, ornamental |
| Elvish (fantasy) | Major modes, Lydian | Bright, ethereal |
| Dwarvish (fantasy) | Aeolian, natural minor, pedals | Heavy, grounded |
| Cosmic/alien | Chromatic, whole-tone, Locrian | Unknowable, disorienting |
| Deep/ancient | Low Aeolian, natural minor, Phrygian | Gravitational, slow |
| Horror/dread | Phrygian, Locrian, diminished, tritone-heavy | Unsettling |
| Oceanic/elemental | Aeolian with suspended harmonies, open 5ths | Vast, flowing |
| Industrial/machine | Chromatic, repetitive minor ostinato | Mechanical, relentless |

See `references/CHARACTER_VOICE_REFERENCE.md` for the complete mode/scale mapping.

### Step 9 -- Write Dialect into Lyrics

**Skip this step for non-vocal entities.** Their sonic identity is expressed through instrumentation and production, not lyric content.

**For vocal characters:**

Choose 5-10 words from the dialect/accent that will appear IN the lyrics:

| Accent | Dialect Words to Use | Effect |
|---|---|---|
| Southern | ain't, gonna, y'all, reckon | Grounds the vocal |
| Irish | aye, lass, grand, fierce | Triggers the lilt |
| Scottish | ye, yer, aye, wee, braw, cannae | Forces rolled R's |
| Appalachian | git, cain't, holler, creek (crik) | Mountain placement |
| Dwarvish-inspired | stone, forge, iron, ale, oath, clan | Grounding words |
| Nordic-inspired | fate, rune, sea, storm, oath, blood | Ancient weight |

**Usage plan:** 1-2 dialect words per verse, more in intimate moments.

**Important:** Dialect in lyrics FORCES the vocal model to commit to the accent. Without dialect words, the style prompt alone may produce inconsistent results.

### Step 10 -- Craft the Suno Style Prompt Descriptor

**For vocal characters -- write the vocal portion of the Style Prompt:**
```
[accent descriptor], [voice quality], [register], [delivery style]
```

**Examples:**
- "Scottish burr accent, rolled R's, deep gravelly baritone, intimate storytelling"
- "ethereal flowing vocal, musical liquid consonants, ancient soprano, reverb-heavy"
- "cold clinical delivery, precise articulation, subtle digital artifacts, female contralto"
- "Southern drawl, warm relaxed phrasing, country inflection"
- "Irish lilt, soft melodic cadence, warm Celtic tone"
- "Nordic vocal style, clear precise vowels, ancient Scandinavian tone"
- "deep male bass baritone dwarven warrior voice, gravelly rough texture with Scottish Nordic burr accent, thick rolled r's"

**For non-vocal entities -- write the sonic presence descriptor:**
```
[primary sound/instrument], [texture quality], [frequency range/spatial character], [temporal behavior], [exclusions]
```

**Examples:**
- "Sub-bass drone below 40Hz, 40-voice choir glossolalia at half-tempo, reversed whale song textures, cathedral-trench reverb, no words, no human language"
- "Crackling static field, machine-breathing rhythm, mid-frequency oscillation, sterile clinical space, no melody, no vocal"

### Step 11 -- Document Voice Evolution (If Album)

If this character appears in multiple tracks:
```
Track X: [voice at this point -- texture, state, delivery]
Track Y: [how it has changed -- what happened, what is different]
Track Z: [final evolution]
```

**Rule:** The voice must be DIFFERENT in every track. Document what changes.

If the album blueprint already documents voice evolution in its Character Voice Registry, reference it rather than duplicating.

Before using the voice design:

- [ ] All applicable template fields filled (mark N/A for non-vocal entities where fields do not apply)?
- [ ] Vowel choices are SINGABLE in the chosen key (vocal characters only)?
- [ ] Instruments match the cultural ecosystem / sonic feeling?
- [ ] Mode/scale matches the culture or emotional territory?
- [ ] Dialect words chosen and placed in lyrics (vocal characters only)?
- [ ] Suno style prompt descriptor written?
- [ ] If album: voice evolution documented?
- [ ] Production space chosen (reverb/room type)?
- [ ] No conflicts with other character voices in the same album?

## Voice Evolution for Albums

When a character appears across multiple tracks, their voice must EVOLVE:

- **Track-to-track evolution:** Each appearance should have at least one audible difference (texture shift, register change, delivery change, production treatment change)
- **Arc-level evolution:** The character's voice should tell its own story across the album (e.g., intimate whisper in T1, full projection in T8, dissolved in T15)
- **Proxemic evolution:** Vocal distance can change to reflect emotional state (Lacasse: intimate to public to dissolved)
- **Never static:** Even subtle changes (slightly more reverb, slightly harsher consonants) prevent staleness
