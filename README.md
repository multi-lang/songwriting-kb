A professional song production system combining Nashville/LA co-writing craft, academic music theory (12 disciplines), Suno AI v5.0 formatting expertise, and concept album continuity management — packaged as a complete Kiro IDE ecosystem.

## Quick Start

Clone this repo. The `.kiro/` directory auto-configures Kiro with 4 agents, 5 skills, 4 hooks, 1 power, and 4 steering files.

```
"Write a song about..."     → Songwriter agent activates
"Critique this song"        → Critic agent scores (12 categories)
"Make this Suno-ready"      → Suno-optimizer adds meta-tags
"Check album continuity"    → Album-continuity verifies rules
```

## Suno AI Formatting System

Battle-tested Suno prompt engineering with 5 formatting modes + 35 confirmed meta-tags:

| Mode | Use Case | Best For |
|---|---|---|
| **A** — Per-Line Vocal Tags | Intimate cinematic dialogue | Complex duets, distinct voices |
| **B** — Parenthetical () | Space-efficient call-and-response | Tight character counts |
| **C** — Theatrical/Concept | Multi-layer concept albums | Production control, custom layers |
| **D** — Character Performance | Fantasy characters, accents | Dialect-driven, worldbuilding |
| **E** — Solo Anthem/Intimate | Accessible solo tracks | Hit formula songs |

### Key Suno Rules
- Style Prompt: ≤1000 characters
- Lyrics field: ≤5000 characters
- Overflow → Advanced box
- Section tags on own lines
- 4-8 lines per section, no end punctuation
- Each line = one melodic phrase (7-12 syllables)

## The 9-Step Professional Workflow

Applied to every song:

1. **SEMANTIC** — What is it literally about?
2. **EMOTIONAL** — Central affect? BRECVEMA mechanisms?
3. **PROSODIC** — Stresses, breaths, syllable counts (7-12 per line)
4. **NARRATIVE** — Arc type? Where is the turn?
5. **VOICE** — Who speaks? Accent? Register?
6. **GENRE** — Fabbri's 5 rules? Fusion ratio?
7. **ARRANGEMENT** — Growth pattern? Contrast? Negative space?
8. **PRODUCTION** — Timbre? Harmonic choices? Space?
9. **COMMERCIAL** — Audience? Platform? Length? Sync?

## Quality Standards

- No line exceeds 12 syllables at the song's tempo
- Hook passes "would someone text this line to a friend?" test
- V2 adds NEW information (not restatement of V1)
- Bridge contains a genuine TURN (meaning shifts)
- Chorus arrives within 45-60 seconds
- ABCB default rhyme scheme (most natural)
- Near/slant rhymes preferred over forced perfect
- Every word earns its place — no filler
- Production choices must SIGNIFY correctly (Tagg semiotics)
