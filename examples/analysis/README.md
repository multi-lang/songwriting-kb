## Using These as Reference

When running the critic on your own songs, compare your output format against these examples. Every critique should follow the template in `references/CRITIQUE_REPORT_TEMPLATE.md`.

The Watching the Clouds v1→v2→v3 progression is the best example of the full revision pipeline in action (SOP 07).
