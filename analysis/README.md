# Analysis Directory

**Your critique reports and analysis outputs go here.** This directory is YOUR workspace.

Example critiques (from the developer's testing) are in `examples/analysis/` for reference.

---

## What Goes Here

| Type | Naming | Example |
|---|---|---|
| Critique reports | `[song-filename]_critique.md` | `My_Song_critique.md` |
| Re-critiques | `[song-filename]_critique_v2.md` | `My_Song_critique_v2.md` |
| Continuity reports | `[song-filename]_continuity.md` | `My_Song_continuity.md` |
| Suno optimization | `[song-filename]_suno_optimization.md` | `My_Song_suno_optimization.md` |
| Album analysis | Descriptive name | `My_Album_Book_Specificity.md` |

## Multi-Album Naming

If you have multiple albums with overlapping track numbers, prefix with an album abbreviation:
- `FS_08_First_Blood_critique.md` (Fractured Shadows)
- `KOTL_01_The_Keepers_Morning_critique.md` (Keeper of the Light)

## Output Format

All critique reports should follow the template in `references/CRITIQUE_REPORT_TEMPLATE.md`.
